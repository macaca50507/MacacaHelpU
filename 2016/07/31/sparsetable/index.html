POJ 3264

題目

使用 Sparse table 針對每一個 query 區間找出最大跟最小值相減就是答案
Sparse Table:
第一個維度 i 是長度( 2 ^ i )，第二個維度 j 是區間開頭位置
想法是透過將一個區間二分，再從前後兩個長度為 2 ^ (i - 1)的區間比較取出最大值
針對每一筆 query 對區間長度取 log 找出兩個可以涵蓋整個 query 的區間再找出最大/小
