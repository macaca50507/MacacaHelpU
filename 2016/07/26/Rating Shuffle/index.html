Rating Shuffle

題目

做法是先二分搜調整的次數 解是一個 0 0 0 1 1 1 1 的分佈

注意上界!!
10^9 是不夠的 最差情況會到 2*10^9 開到 10^10 就過了

進入判斷函式針對每一個 s[i] s[i - 1] 找出一個最佳的調整次數 cnt
針對 本次二分搜的 k 值 與 cnt 比較有兩種

1.奇偶性 2.兩者大小關係
同奇偶的狀況可以透過 2 次震盪來消耗掉 2 的倍數次

一個有趣的觀察 :
唯一 return false 的情況是沒辦法調成比前項小

再分成幾種 CASE :

舉個例子 :

  1. 當你要變得比前項大且 k >= cnt 時，如果同奇偶那直接選cnt就好
    (k - cnt) 這些多餘的調整次數可以透過來回震盪來抵銷掉
    如果不同奇偶那只能退而求其次選 (cnt - 1)
    重要的觀察是如果不同奇偶，多餘的次數會是奇數沒有辦法震盪抵銷掉
    但如果是 (cnt - 1) 這個 -1 會造成奇數變成偶數 or 偶數變成奇數
    就又可以讓你透過來回震盪用光次數了

  2. 當你要變得比前項大且 k < cnt 時，那直接用 k 就好，用光就對了!
    cnt 是最優調整次數 你都不足 cnt 了，當然是調好調滿也沒有多餘次數的問題

  3. 減法的時候就自己推一下吧 ~
